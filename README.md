Apache Spark Learning Repository

Overview

This repository contains my hands-on practice and learning materials for Apache Spark using PySpark.
The goal is to understand Spark fundamentals, DataFrame transformations, and distributed data processing.

Technologies Used

- Python
- Apache Spark (PySpark)
- Jupyter Notebook
- Docker
- Git & GitHub

Project Notebooks

1. Spark Session

File: "01_Spark_Session.ipynb"

Topics covered:

- Creating a Spark Session
- Understanding Spark Context
- Running basic Spark commands

2. Basic Transformations

File: "02_Basic_Transformations_1.ipynb"

Topics covered:

- select()
- filter()
- withColumn()
- drop()
- rename()

3. Spark SQL / DataFrame Operations

File: "03_Spark_SQL_DataFrame.ipynb"

Topics covered:

- Working with Spark DataFrames
- Basic SQL queries in Spark
- DataFrame operations

Learning Objective

The purpose of this repository is to build a strong foundation in Apache Spark before moving to advanced topics like:
